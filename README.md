# @amlplugins/google-cloud-run

Native `@google-cloud/run` SDK re-exported under the `@amlplugins` namespace.

> Cloud Run.

## Install

```bash
npm install @amlplugins/google-cloud-run
```

## Use

```ts
import { /* named exports from @google-cloud/run */ } from "@amlplugins/google-cloud-run";
// or
import sdk from "@amlplugins/google-cloud-run";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
